# 5.10 thisのキャプチャ

## 5.10.1 メンバー変数のキャプチャ
メンバー関数の中でラムダ式を使う場合、メンバー変数をキャプチャできないという問題があります。
例えば次のように、メンバー変数をコピーでキャプチャに指定してもエラーとなります。これは参照のキャプチャであっても同様です。

```C++
class C {
  int a; // キャプチャしたいメンバー変数

public:
  void show_a() {
    [a]() { // エラー。メンバー変数を直接キャプチャできない
      std::cout <<"C::a = " << a << std::endl;
    }();
  }
};
```
